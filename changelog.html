Changelog

Complete version history and updates for PDF Mate

v1.2.1 April 2026 Minor
✨ New
  • Now available on both Chrome Web Store and Microsoft Edge Add-ons
  • Automatic browser detection for store links
🔧 Improved
  • Enhanced format conversion stability for Word and PowerPoint exports
  • Improved memory handling during large file processing
  • Updated UI components for better accessibility
🐛 Fixed
  • Resolved an issue where cleared files would reappear after reopening the extension
  • Fixed PDF rotation type errors in TypeScript strict mode
  • Corrected pricing display inconsistencies in landing page
v1.2.0 March 2026 Minor
✨ New
  • Added comprehensive Help Center with FAQ section
  • New changelog section in landing page
🔧 Improved
  • Refactored storage layer for better performance
  • Enhanced error handling across all PDF operations
v1.1.0 February 2026 Major
✨ New
  • Convert PDF to Word — Export PDFs as editable .docx documents
  • Convert PDF to PowerPoint — Transform PDFs into .pptx presentations
  • Convert PDF to Images — Export pages as PNG or JPG with adjustable quality and DPI (72-300)
  • Added file staging — uploaded files are temporarily cached locally
🔧 Improved
  • Better memory handling for large PDF files during conversion
  • Enhanced dark mode theme consistency
  • Improved drag-and-drop file upload experience
v1.0.1 January 2026 Patch
🔧 Improved
  • Optimized PDF rendering performance
  • Reduced extension bundle size
🐛 Fixed
  • Fixed theme switching flicker on initial load
  • Corrected page count display for certain PDF formats
v1.0.0 December 2025 Major
✨ New
  • Initial release — PDF Mate is now available on Chrome Web Store
  • Merge PDFs — Combine multiple PDF files with drag-and-drop ordering
  • Split PDF — Extract pages by individual, ranges, or fixed chunks
  • Rotate Pages — Rotate 90°, 180°, 270° clockwise or counter-clockwise
  • Delete Pages — Remove unwanted pages with visual selection or custom ranges
  • Reorder Pages — Rearrange page order with intuitive controls
  • Free & Pro Tiers — Free tier with up to 10 pages; Pro with unlimited access
  • Dark Mode — Full support for light, dark, and system themes
  • Bilingual UI — English and Chinese language support
  • 100% Offline — All processing happens locally in your browser
